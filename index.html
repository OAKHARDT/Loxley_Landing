The Problem

Creative worlds often end up scattered across notes, folders, chats, spreadsheets, images, maps, and AI prompts. LOXLEY is being built to bring that material into one structured, portable world system.

What LOXLEY Is

LOXLEY is a creative worldbuilding app for organizing fictional universes. It helps creators manage characters, locations, factions, relationships, timelines, maps, lore, and visual references in one connected workspace.

At the centre of LOXLEY is the .lxw file — a portable world package designed to preserve the structured truth of a creative project.

The .lxw World File

.lxw is being designed as LOXLEY's portable world package — a structured container for the people, places, maps, lore, relationships, assets, and rules that make up a fictional universe.

The goal is simple: build a world once, then use it across apps, readers, websites, games, AI workspaces, and future creative tools.

Portable

Your .lxw world file is yours to keep, use, and share. Not trapped in a single app or cloud service.

Structured

The .lxw format is a well-defined package for storing creative world data in a machine-readable way.

Future-Ready

AI tools, game engines, website generators, and readers can all work with .lxw worlds — without losing data.

Local-First

Your world lives on your device first. Cloud features may come later, but the foundation is local ownership.

What is a .lxw file?
A .lxw file is LOXLEY's planned portable world package — a structured file for storing creative world data. It's designed to work across apps, readers, websites, games, and AI tools.

The goal is local-first ownership. Cloud features may come later, but the foundation is being built around local files and user control.
Who is LOXLEY for?
Writers, worldbuilders, RPG creators, indie game developers, comic creators, and anyone building a fictional universe.
Will AI be involved?
The long-term plan is for AI tools to safely read from .lxw worlds and help generate projects (comics, games, scripts, reports), while LOXLEY remains the source of canon.
